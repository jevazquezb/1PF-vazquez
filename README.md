## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    Git & Github
    Node (NVM & NPM)
    Json-server
    Angular
    

### Setup

    1. git clone https://github.com/jevazquezb/1PF-vazquez.git
    2. cd 1PF-vazquez
    3. npm install
    4. json-server --watch db.json

### Usage

    1. ng serve -o
